# 🌍 WorldGuessr God Mode Hack

**Advanced automated bot for WorldGuessr with 100% accuracy.**
Supports **Singleplayer**, **Multiplayer**, **Community Maps**, and **Private Games**.

## Features

- 🎯 **Pinpoint Accuracy**: Automatically detects the exact coordinates of the current round.
- 🔴 **Visual Indicator**: Draws a **50km radius red circle** on the in-game map around the target location.
- 🕵️ **Stealth Mode**: No visible HUD or overlays, just the map marker.
- 🌐 **Universal Support**: Works on all game modes, including custom community maps.
- 🚀 **Performance**: Optimized to run smoothly with minimal overhead.

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/PresiDeWitt/worldguessr-hack.git
    cd worldguessr-hack
    ```

2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the script**:
    ```bash
    python worldguessr_god.py
    ```

## Usage

1.  **Run the script**:
    ```bash
    python3 worldguessr_god.py
    ```

2.  **Play**:
    - A Chrome browser window will open automatically.
    - Navigate to [WorldGuessr](https://www.worldguessr.com/).
    - Start any game mode.
    - The hack will automatically detect the location and draw the red circle on the map.
